# Proyecto Módulo 1 – Fundamentos

## Descripción
Aplicación desarrollada en **Streamlit** que integra los conceptos fundamentales de la programación en Python.  
El proyecto consolida el aprendizaje del **Módulo 1 – Python Fundamentals** de la Especialización en Python for Analytics, aplicando de manera práctica:

- Variables y tipos de datos  
- Condicionales  
- Estructuras de datos (listas, diccionarios)  
- Funciones  
- Programación funcional  
- Programación Orientada a Objetos (POO)  

La aplicación permite registrar y analizar actividades financieras, evaluar presupuestos y visualizar resultados de forma interactiva mediante una interfaz web.
